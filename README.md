# React.js

Repositório de estudos desenvolvido durante uma certificação de React.js.

## Sobre o repositório

Este repositório reúne exercícios, aulas e práticas realizadas durante uma certificação de 40 horas em React.js.

O objetivo foi estudar os principais conceitos da biblioteca, como componentes, props, estados, eventos, rotas, formulários, estilização e organização de projetos React.

Este repositório funciona como registro da minha evolução durante a certificação.

## Conteúdos estudados

- Introdução ao React.js
- Componentes
- Props
- Estados
- Eventos
- Renderização de listas
- Formulários
- Rotas
- Estilização
- Organização de projetos
- Práticas com conceitos fundamentais e avançados

## Tecnologias utilizadas

- React.js
- JavaScript
- HTML
- CSS
- Node.js / npm

## Como executar os projetos

1. Clone este repositório:

```bash
git clone https://github.com/JohnElthon/React.js.git
```

2. Acesse a pasta do projeto/aula desejada.

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm start
```
